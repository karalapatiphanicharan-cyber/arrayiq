# Algorithm Documentation

## Sorting Algorithms
| Algorithm | Average Case | Space Complexity | Type |
|-----------|--------------|------------------|------|
| Quick Sort | O(n log n) | O(log n) | Divide & Conquer |
| Merge Sort | O(n log n) | O(n) | Divide & Conquer |
| Heap Sort | O(n log n) | O(1) | Selection |
| Insertion Sort | O(n²) | O(1) | Insertion |
| Bubble Sort | O(n²) | O(1) | Exchanging |
| Selection Sort | O(n²) | O(1) | Selection |

## Searching Algorithms
| Algorithm | Complexity | Requirements |
|-----------|------------|--------------|
| Linear Search | O(n) | None |
| Binary Search | O(log n) | Sorted Array |
| Jump Search | O(√n) | Sorted Array |
| Interpolation Search | O(log log n) | Sorted Numeric Array |

## Quantum Simulations
- **Grover's Algorithm**: Theoretical speedup for searching unsorted databases (O(√n) vs O(n)).
- **Quantum Walk**: Generalization of classical random walks with speedup potential.
